# Gerador de Senha Simples

Este projeto é um **gerador de senha simples**, desenvolvido com **HTML, CSS e JavaScript** em arquivos separados.

A proposta é oferecer um exemplo pequeno, organizado e fácil de compreender, ideal para professores utilizarem como modelo em aulas de introdução à criação de páginas web e programação.

---

## Objetivo do projeto

O objetivo deste projeto é demonstrar, de forma prática, como criar uma página interativa usando três tecnologias básicas da web:

- **HTML** para estruturar o conteúdo da página;
- **CSS** para definir a aparência visual;
- **JavaScript** para adicionar interação e gerar a senha automaticamente.

---

## Como funciona

Ao abrir a página, o usuário verá um campo de texto e um botão.

Quando o botão **Gerar senha** é clicado, o JavaScript cria uma senha aleatória com letras maiúsculas, letras minúsculas, números e símbolos.

A senha gerada aparece automaticamente no campo da página.
